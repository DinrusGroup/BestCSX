Example

If a call to this method is not already wrapped in a transaction, then this call will execute within it's own transaction.

Example using ForAll to perform some action against the record set.

C#
db.ForAll<Customer>(
	c => Action(c)
);

Manual way of acheiving the same results as above.

C#
var results = db.GetAll<Customer>();
foreach (var r in results)
	Action(r);
