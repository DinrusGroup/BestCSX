Remarks

All types from mscorlib.dll that implement the IConvertable interface are supported, and also TimeSpan, Guid and Nullable<T> where T is also a supported type.

The following is the list of supported types.
  Boolean
  Byte
  Char
  DateTime
  DateTimeOffset
  Decimal
  Double
  Enum
  Int16
  Int32
  Int64
  SByte
  Single
  String
  UInt16
  UInt32
  UInt64
  TimeSpan
  Guid
  Nullable<T> where T is a supported type.
