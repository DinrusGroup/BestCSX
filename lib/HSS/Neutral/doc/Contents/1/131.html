Example

Example showing how to use the Cast extension method. Behind the scenes, the .NET LINQ Query internals will search for and use this Cast extension method, which simply calls database.Query<T>().

Note the Model casting (from Model m in ...) in the first query expression.
C#
var query = from Model m in database
            				where m.Age > 30
            				select m;
            
// Functional equivalent without requiring a call to the Cast extension method.
var query = from m in database.Query<Model>()
            				where m.Age > 30
            				select m;
